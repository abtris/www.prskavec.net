You need to consider a few things if you have primary and secondary.

- Will you rotate the same people on both rosters?
- Should more junior engineers be on primary, with more experienced engineers on secondary?
- If people rotate on both rosters, what happens if both people don’t know what to do?
- How do we escalate if both levels fail? Do we need a manager on-call? Should we page all team members instead?
